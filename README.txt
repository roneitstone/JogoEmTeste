# JogoEmTeste

Versão compilada do jogo para Linux.

O executável é LunarLander-Xac.x86_64

Atenção, o executavel precisa ter permissão para funcionar em ambiente Linux. Caso o contrário o usuário terá problemas para iniciá-lo.

Aviso: O jogo e a pesquisa ainda está em desenvolvimento, logo algumas funções estão incompletas.
